## JS-Link


This Discord bot handles Donators and has a link feature for osu private server.

## Requirements
- NodeJS 14.16
- MySQL
- Redis

## Setting up

install the dependencies

```
$ npm install
```

fill out the config.json file
```
$ nano config.json
```

start the bot
```
$ node main.js
```
